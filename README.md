# Roamly

AI travel planner — enter a city, budget, group size, and interests, and get a full day-by-day itinerary with photos, menus, weather, and Google Maps routes.

**[Full documentation →](./roamly/README.md)**

## Quick start

```bash
cd roamly
npm install
cp .env.example .env.local   # then add your API keys
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
